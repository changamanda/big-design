Switch

Switches are a stylized input[type="checkbox"] with controllable checked/unchecked states. Switches are intended to be used for immediate toggle actions and are therefore not intended to be used in forms.

Edit the code below to see your changes live!

function Example() {
  const [checked, setChecked] = useState(false);
  const handleChange = () => setChecked(!checked);
  return <Switch checked={checked} onChange={handleChange} />;
}
